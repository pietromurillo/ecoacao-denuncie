EcoAção – Plataforma de Denúncias Ambientais

Este é um projeto simples onde qualquer pessoa pode registrar denúncias ambientais enviando descrição, localização e uma foto. As denúncias aparecem automaticamente na página inicial.

Para rodar o projeto, é só seguir estes passos:

Instale o Node.js no seu computador.

Baixe ou clone o projeto.

No terminal, abra a pasta do projeto.

Execute o comando npm install para instalar as dependências.

Depois rode node server.js.

Abra http://localhost:3000
 no navegador.

As imagens enviadas ficam salvas na pasta uploads, e as denúncias ficam guardadas no arquivo db.json automaticamente.
